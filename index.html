About NBA Salary Dashboard

This dashboard provides comprehensive visualization of NBA player salary data, organized by teams and players. The data is collected using web scraping techniques from public NBA salary databases.

Features:

  • Interactive salary distribution charts
  • Team payroll comparisons
  • Player salary rankings
  • Statistical analysis and insights
  • Data export functionality
